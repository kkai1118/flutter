# 01 补充：Flutter 是什么

对应主文档章节：`1. Flutter 是什么`

## 先建立一个整体印象

Flutter 不是单纯的组件库，而是一整套跨平台 UI 开发方案。

- 用 Dart 编写
- 用 Widget 描述界面
- 一套代码可以构建多端应用

## 你真正要学的不是“组件名字”

初学者最容易误以为 Flutter 就是背组件。

其实更重要的是：

- 页面怎么由 Widget 组合出来
- 数据变化后界面为什么会刷新
- 页面之间怎么跳转
- 布局为什么会报错或溢出

## 学 Flutter 的正确预期

- 先能看懂基础页面
- 再能自己拼出简单页面
- 最后再进入状态管理、网络请求、工程化结构
